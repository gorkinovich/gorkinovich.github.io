Apuntes antiguos de programación

Siendo estudiante de informática tuve que aprender todo tipo de lenguajes. Para facilitar consultas posteriores, una buena práctica era hacer apuntes de los lenguajes que iba aprendiendo. Como resultado realicé los siguientes apuntes: C++, C#, Haskell, Java 2, Lisp, OpenGL, Pascal, PHP y MySQL, Prolog, Scala, Unix.

También en su día redacté una guía breve para aprender cómo se tiene que realizar el documento de diseño para un videojuego. Básicamente el documento de diseño es la biblia que contiene todos los detalles sobre el funcionamiento del juego en cuestión. Es muy importante dejar documentado dicho funcionamiento, para que el equipo lo pueda consultar y todos remen en la misma dirección.
